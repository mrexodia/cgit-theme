#cgit-theme

A theme for cgit

## Setup
Simply copy all files to your cgit root folder make the reflective changes in 
your cgit configuration `/etc/cgitrc`.

```
css=/local-cgit.css
logo=/git-logo.png
favicon=/git-favicon.png
```
